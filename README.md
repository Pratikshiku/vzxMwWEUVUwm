- 后端：`Java`、`JavaWeb` / `Springboot`。

### 三、功能描述
系统分为（身份）：普通用户、医生、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 医生
5. 公告信息
6. 测试模板
7. 个人中心
8. 考试记录
9. 错题本
10. 修改密码
11. 个人信息
12. 咨询信息管理
13. 诊断信息管理


*医生功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 公告信息管理
6. 咨询信息管理
7. 诊断记录管理


**管理员功能：**
1. 登录
